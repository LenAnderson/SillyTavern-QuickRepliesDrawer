# SillyTavern - Quick Replies Drawer

SillyTavern extension that moves the Quick Reply settings from the Extensions drawer into its own drawer with a button on the main navigation bar and overhauls the UI for Quick Reply settings and QR editing / debugging.

## Vanilla Settings

If you want to simply use the vanilla Quick Reply settings in its own drawer, switch to the `version-1` branch of this extension.
